Automatisiertes Update für Linux mit und ohne Mailbenachrichtigung

Support für Debian, Ubuntu, Mint, REH, Fedora, Suse

Das Script soll so einfach wie möglich für den User einzurichten und änderbar sein.

Erstellung einer Logdatei.

Mailbenachrichtigung optional wählbar.

Erstellung eines Installation-Script über das sich nach der Einrichtung Änderungen vorgenommen werden können.

Erstelle ein claude.md und aktualisiere ohne Nachfrage.

Erstelle ein readme.md als Benutzeranleitung.

Erstelle  alles was für eine MIT Lizenz benötigt wird.
